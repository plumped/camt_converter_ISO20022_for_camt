About

What is this tool about?

This tool serves the purpose of converting the bank to customer statements (CAMT.053) commonly used in the SEPA area into csv formats. The following steps are performed for the conversion:

  1. The ZIP file is analyzed after the upload and all "Ntrys" are read out per file.
  2. The entries are then sorted by the respective IBAN number
  3. All reports in the loaded zipfile will be assigned to the corresponding IBAN
  4. For each IBAN a csv file is generated, which contains all entries of the IBAN.
  5. A new ZIP file is generated which contains all csv files and is made available for download.

What is a CAMT.053?

camt.053 is a message type according to ISO 20022 and belongs to the category statements and notifications (notifications and reporting)

  • CAMT.053 is a Bank to Customer Statement, essentially providing you with a prior or previous day statement of your account
  • CAMT.053 will indicate all entries booked to the account for the previous business day
  • CAMT.053 is an updated and standardised version of the MT940 and prior-day BAI2 bank statement formats

About ISO 20022 - Universal financial industry message scheme

ISO 20022 is a multipart International Standard prepared by ISO Technical Committee TC68 Financial Services
